# Proofworks public site (GitHub Pages) — v2

This folder is a complete static website you can drop into your `proofworks.us` GitHub Pages repo.

## What it does

- Professional dark landing page
- **Two equal “paths”** (Trade / Oddities) that route into the app using:
  - `https://app.proofworks.us?path=trade`
  - `https://app.proofworks.us?path=oddity`
- A separate **recognition landing page** for your DOT bracelet tap:
  - `/recognition/`
- Legal pages:
  - `/terms/` `/privacy/` `/steward-agreement/` `/refunds/`

## Quick deploy (GitHub UI)

1. Open the GitHub repo that hosts **proofworks.us** (GitHub Pages).
2. Click **Add file → Upload files**.
3. Drag and drop the contents of this folder **into the repo root**:
   - `index.html`
   - `assets/`
   - `recognition/`
   - `how-it-works/`
   - `contact/`
   - `terms/`
   - `privacy/`
   - `steward-agreement/`
   - `refunds/`
4. Click **Commit changes**.
5. Wait a minute, then visit `https://proofworks.us`.

## Customize

- Update the app URL if needed (search for `https://app.proofworks.us`).
- Replace `meme1.png` / `meme2.png` in `assets/images/` with your preferred graphics.
- Update the contact email in `/contact/index.html`.
